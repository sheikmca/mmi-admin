# mmi-admin

## Requirement 
* openjdk:11
* spring-boot 2.6.2


## Introduction 
* mmi-admin contaions 3 parts
  1. mmi-admin-server is a server provide management features of User, role, UserGroup and so on. 
  2. mmi-admin-pojo is a jar provide input or output object definition of Restful Api.  
  3. mmi-admin-feign provide Feign feature for mmi-admin interface.
